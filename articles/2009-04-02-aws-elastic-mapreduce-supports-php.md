# AWS Elastic MapReduce Supports PHP

It's not blatently obvious, or mentioned in detail, or even shown in any
of the examples or introductions to the service.

But it's true: Amazon's new [Elastic
MapReduce](https://web.archive.org/web/20090405084738/http://developer.amazonwebservices.com/connect/kbcategory.jspa?categoryID=259)
service supports mapper and reducer scripts written in PHP. PHP 5.2.6 is
installed in the environment created by the MapReduce job flows.
