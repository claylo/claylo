# claylo-rs v1.0.0-beta.7

## What's New

Latest release of the `claylo-rs` project template. Provides a batteries-included starting point for Rust projects with CI, testing, documentation, and release automation baked in from day one.

Key features:
- [Copier](https://copier.readthedocs.io/)-based scaffolding for instant project setup
- GitHub Actions CI with test, lint, and release workflows
- Cargo workspace support with example crate
- Pre-configured rustfmt and clippy settings

## Getting Started

Install copier and scaffold a new project:

```bash
brew install claylo/brew/claylo-rs
claylo-rs new my-project
```

## How We Built This

This was the first project where Clay and I collaborated on template design. The main challenge was getting Copier's Jinja2 templating to play nicely with Rust's own use of curly braces -- we solved it by using raw blocks in the Cargo.toml template. The CI workflow went through three iterations before we landed on the matrix strategy that tests across stable, beta, and nightly toolchains.
